# 🎡 Game Roulette

A fun and interactive web-based game roulette spinner where you can discover random games to play!

## Features

- **Interactive Wheel**: Visually stunning roulette wheel with 8 classic games
- **Smooth Animations**: Satisfying spin animation with realistic physics
- **Statistics Tracking**: Keeps track of which games you've been assigned
- **Local Storage**: Your statistics are saved in your browser
- **Play Integration**: Click "Play Game" to open the game in a new tab
- **Responsive Design**: Works perfectly on desktop and mobile devices

## Games Included

1. **Tetris** - Classic block-stacking puzzle game
2. **Chess** - The timeless strategy board game
3. **Snake** - Retro arcade snake game
4. **Pong** - Vintage paddle game
5. **Pac-Man** - Classic maze chase game
6. **Flappy Bird** - Casual tap-to-play game
7. **2048** - Number puzzle game
8. **Sudoku** - Logic puzzle game

## How to Use

1. Open `index.html` in your web browser
2. Click the **"SPIN!"** button to spin the roulette wheel
3. Watch the animation and wait for the result
4. Click **"Play Game"** to open the selected game
5. Your statistics are automatically saved and displayed

## Files

- `index.html` - Main HTML structure
- `styles.css` - Styling and animations
- `script.js` - Game logic and interactions
- `README.md` - Documentation

## Features Explained

### Wheel Spinner
The wheel spins with a smooth animation and randomly selects one of 8 games.

### Statistics
Track how many times each game has been selected. The stats are sorted by frequency and stored in your browser's local storage.

### Responsive Design
Works on all devices - desktop, tablet, and mobile.

## Customization

You can easily customize the games:

1. Edit the `games` array in `script.js` to change game names
2. Update the `gameLinks` object with your preferred game URLs
3. Modify the colors in `styles.css` for a different aesthetic
